# HSK 1-3 学习助手

交互式网页应用程序，用于学习HSK 1、2、3级汉语词汇。

## 功能特点

- **三个学习级别**: HSK 1、HSK 2、HSK 3
- **四种学习模式**:
  - 学习新词汇（卡片翻转）
  - 复习已学词汇
  - 词汇测试
  - 所有词汇列表
  
- **进度跟踪系统**:
  - 分别跟踪每个级别的学习进度
  - 进度条显示
  - 本地存储保存进度
  
- **中文主题设计**:
  - 柔和的中文风格配色
  - 响应式设计
  - 用户友好界面
  - 汉字装饰元素

## 使用方法

1. 打开 `index.html` 在浏览器中
2. 选择HSK级别 (1, 2, 3 或 全部)
3. 从"学习新词"模式开始
4. 点击卡片查看翻译和例句
5. 使用"已学会"按钮标记掌握的词汇
6. 在"复习"模式中巩固已学词汇
7. 在"测试"模式中检查掌握程度
8. 在"所有词汇"模式中查看完整列表

## 键盘快捷键（在学习模式中）

- **空格键** - 翻转卡片
- **左箭头** - 上一张卡片
- **右箭头** - 下一张卡片
- **L** - 标记为"已学会"
- **S** - 随机打乱卡片
- **1/2/3/A** - 切换到HSK 1/2/3/全部级别
