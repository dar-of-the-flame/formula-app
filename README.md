# HSK 1 Learning App

Интерактивное веб-приложение для изучения 150 слов из HSK 1 (первый уровень китайского языка).

## Функции

- **Три режима обучения**:
  - Изучение новых слов (карточки с иероглифами)
  - Повторение изученных слов
  - Тестирование знаний
  
- **Система прогресса**:
  - Отслеживание изученных слов
  - Прогресс-бар
  - Сохранение в localStorage
  
- **Китайский дизайн**:
  - Тематическое оформление в китайском стиле
  - Адаптивный дизайн
  - Удобный интерфейс

## Использование

1. Откройте `index.html` в браузере
2. Начните с режима "Изучение"
3. Кликайте на карточки, чтобы перевернуть и увидеть перевод
4. Отмечайте изученные слова кнопкой "Изучено"
5. Повторяйте слова в режиме "Повторение"
6. Проверяйте знания в режиме "Тестирование"

## Горячие клавиши (в режиме изучения)

- **Пробел** - перевернуть карточку
- **Стрелка влево** - предыдущая карточка
- **Стрелка вправо** - следующая карточка
- **L** - отметить как изученное
- **S** - перемешать карточки
